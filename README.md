🍲 Food Donation Management System
📌 Overview

The Food Donation Management System is a frontend-based web application designed to reduce food waste by connecting food donors with administrators who manage and track donations. Users can add surplus food details, and admins can view, manage, and update donation statuses.

This project is built using HTML, CSS, and JavaScript, and uses localStorage to simulate backend data persistence.

🚀 Live Demo

👉  https://sudhaharini05.github.io/FoodDonationManagementSystem/

🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
LocalStorage (for data persistence)
🎯 Features
👤 User Module
Add food donation details
View added donations
Track donation status
🛡️ Admin Module
View all donations
Mark donations as “Claimed”
Delete invalid entries
Manage donation records
💾 Data Handling
Uses browser localStorage to store and retrieve donation data
Data persists even after page refresh

⚙️ How to Run Locally
Download or clone the repository:
git clone https://github.com/SudhaHarini05/food-donation-system.git
Open the folder in VS Code
Install Live Server extension
Run:
Right click index.html
Click Open with Live Server
🌐 Deployment (GitHub Pages)

This project is deployed using GitHub Pages.

Steps:

Push code to GitHub repository
Go to Settings → Pages
Select branch: main
Select folder: /root
Save and get live URL
